# Cara Menambahkan Gambar

## Langkah-langkah:

1. Simpan semua gambar produk Anda ke folder `public/images/`

2. Beri nama file gambar sesuai kebutuhan, contoh:
   - `hero-1.jpg`, `hero-2.jpg`, `hero-3.jpg`, `hero-4.jpg` (untuk hero section)
   - `product-1.jpg`, `product-2.jpg`, `product-3.jpg` (untuk section elevate)
   - `showcase.jpg` (untuk section designed)
   - `featured.jpg` (untuk section featured product)

3. Gambar akan otomatis muncul di website

## Struktur Folder:
```
public/
  images/
    hero-1.jpg
    hero-2.jpg
    hero-3.jpg
    hero-4.jpg
    product-1.jpg
    product-2.jpg
    product-3.jpg
    showcase.jpg
    featured.jpg
```

## Format yang Disarankan:
- Format: JPG atau PNG
- Ukuran hero: 400x600px (portrait)
- Ukuran product: 400x600px (portrait)
- Ukuran showcase: 600x800px (portrait)
- Ukuran featured: 600x800px (portrait)

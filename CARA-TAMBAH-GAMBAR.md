# Cara Menambahkan Gambar

## Langkah-langkah:

1. Simpan semua gambar produk Anda ke folder `public/images/`

2. Beri nama file gambar sesuai kebutuhan:

### Halaman Home:
   - `hero.jpg` (untuk hero section - 1 gambar saja)
   - `product-1.jpg`, `product-2.jpg`, `product-3.jpg` (untuk section elevate)

### Halaman About Us:
   - `about-hero.jpg` (untuk hero section about)
   - `about-story.jpg` (untuk section Our Story)
   - `about-mission.jpg` (untuk section Our Mission)

### Halaman Collections:
   - `collection-1.jpg` sampai `collection-9.jpg` (9 gambar produk)

3. Gambar akan otomatis muncul di website

## Struktur Folder:
```
public/
  images/
    # Home Page
    hero.jpg
    product-1.jpg
    product-2.jpg
    product-3.jpg
    
    # About Page
    about-hero.jpg
    about-story.jpg
    about-mission.jpg
    
    # Collections Page
    collection-1.jpg
    collection-2.jpg
    collection-3.jpg
    collection-4.jpg
    collection-5.jpg
    collection-6.jpg
    collection-7.jpg
    collection-8.jpg
    collection-9.jpg
```

## Format yang Disarankan:
- Format: JPG atau PNG
- Ukuran hero: 1920x1080px (landscape)
- Ukuran product: 400x600px (portrait)
- Ukuran about images: 800x1000px (portrait)
- Ukuran collection: 400x600px (portrait)

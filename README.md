# Afework Moments Website

Website fashion brand Afework Moments dibuat dengan Next.js 14 (App Router) dan Tailwind CSS.

## Cara Menjalankan

1. Install dependencies:
```bash
npm install
```

2. Jalankan development server:
```bash
npm run dev
```

3. Buka browser di [http://localhost:3000](http://localhost:3000)

## Teknologi

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React 18

## Fitur

- Responsive design
- Modern UI/UX
- Hero section dengan overlay
- Product showcase
- Feature highlights
- Smooth scrolling
